# Habitat Suitability Model – Rock ptarmigan Lagopus muta

## Overview
This GIS project models habitat suitability for the Rock ptarmigan in the national Park Hohe Tauern in Austria using environmental variables.

## Data
- Digital Elevation Model (SRTM)
- Land cover data (Copernicus)
- Road network (OpenStreetMap)

## Methods
- Raster normalization
- Weighted overlay analysis
- Spatial analysis in QGIS

## Tools
- GDAL
- Raster Calculator

## Output
Habitat suitability map showing optimal areas for species distribution.
